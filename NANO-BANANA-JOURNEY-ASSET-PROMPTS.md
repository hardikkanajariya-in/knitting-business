# Nano Banana prompts — About Journey assets

Use the attached reference image as the **primary visual reference** for composition, illustration style, colors, line weight, and the slightly hand-drawn premium infographic feel.

## Global instruction for every prompt

- Match the exact visual language of the attached reference image.
- Preserve the clean editorial infographic style: white / light-gray background, thin gray line work, soft blue + red + beige accents, subtle shading, no harsh realism.
- Output **transparent background PNG** unless SVG/vector is explicitly supported.
- Keep edges clean and isolated.
- No text, no numbers, no watermark, no background scene.
- Center the asset with comfortable padding so it can be placed into a webpage.
